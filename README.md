## Diabetic Retinopathy Detection using Deep Learning

## Overview
This project detects Diabetic Retinopathy from retinal fundus images using a hybrid deep learning model based on EfficientNetB0 and MobileNetV2.

## Technologies
- Python
- TensorFlow
- OpenCV
- NumPy
- Pandas
- Scikit-learn

## Features
- Image preprocessing
- Hybrid CNN Model
- Five-class classification
- High validation accuracy
